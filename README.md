# 🧮 Neon Calculator

A modern and visually appealing **Neon Calculator** built using **HTML, CSS, and JavaScript**. The application performs basic arithmetic calculations with a stylish neon-themed user interface and keyboard support.

---

## 🚀 About the Project

Neon Calculator is a responsive web application designed to perform mathematical calculations in a simple and interactive way. The project combines modern UI design with essential calculator functionality, making it a great beginner-friendly frontend project.

---

## ✨ Features

✅ Addition, Subtraction, Multiplication, and Division

✅ Percentage Calculations

✅ Decimal Number Support

✅ Clear (C) Button

✅ Delete Last Character (DEL)

✅ Keyboard Support

✅ Error Handling for Invalid Expressions

✅ Responsive Neon-Themed Design

---

## ▶️ How to Run

### Step 1

Download or clone the repository.

### Step 2

Open the project folder in VS Code.

### Step 3

Open `index.html` in your browser.

### Step 4

Start performing calculations using either:

* Calculator Buttons
* Keyboard Input

---

## ⌨️ Keyboard Shortcuts

| Key       | Action                |
| --------- | --------------------- |
| 0 - 9     | Enter Numbers         |
| + - * /   | Arithmetic Operators  |
| Enter     | Calculate Result      |
| Backspace | Delete Last Character |
| Escape    | Clear Display         |
| .         | Decimal Point         |

---

## 🧾 Available Operations

### ➕ Addition

Add two or more numbers.

### ➖ Subtraction

Subtract numbers easily.

### ✖️ Multiplication

Perform multiplication operations.

### ➗ Division

Perform division calculations.

### 💯 Percentage

Calculate percentages using the % operator.

### 🗑️ Delete

Remove the last entered character.

### 🧹 Clear

Reset the display instantly.
